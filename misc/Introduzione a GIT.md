Git è un sistema di controllo versione distribuito, usato per tracciare le modifiche nel codice sorgente durante lo sviluppo. Permette a più sviluppatori di lavorare contemporaneamente su un progetto, mantenendo una cronologia completa delle modifiche, facilitando il lavoro collaborativo e il recupero di versioni precedenti.
# Installazione su Windows

- Scarica e installa GIT dal sito o dalla [repository ufficiale](https://gitforwindows.org/)
    Clicca sempre **Avanti** ad ogni scelta durante il setup

## Creare una repository

- Usa **Esplora Risorse** di Windows per navigare verso la cartella radice del progetto che vuoi tracciare usando GIT
- Fai click destro e scegli **Apri nel Terminale**, questo dovrebbe aprire **Windows Powershell** nella cartella scelta.
- Scrivi `{sh}git init` e premi **Invio** per creare la repository
